# Contributing

Thanks for helping improve FileEquinox.

## Getting Started

1. Fork the repository and clone your fork.
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the app locally:
    ```bash
    npm start
    ```

## Development Guidelines

- Keep changes focused and easy to review.
- Use clear commit messages.
- Prefer small, well-tested updates.
- If you add new features, update the documentation where appropriate.

## Formatting

If Prettier is available in the project, format your changes before submitting:

```bash
npm run format
```

## Pull Requests

- Describe what changed and why.
- Include any relevant testing steps.
- Make sure the app still starts after your changes.
